# Tuner

[Player index](README.md) · [Getting Started](getting-started.md) · [Forge](forge.md)

The Tuner is a timing-based upgrade station. You insert the required shard and the item you want to tune, then complete the resonance sequence.

## Basic flow

1. Craft and place a Tuner
2. Place the required shard in the shard slot
3. Place the item to upgrade in the item slot
4. Start the sequence
5. Complete the required rounds
6. Claim the result

## Important notes

- the exact shard and item combinations depend on server recipes
- failed attempts may return a fallback item instead of the desired result
- some recipes are faster or have fewer rounds than others
- the server can disable the debug slot entirely
