# Forge

[Player index](README.md) · [Getting Started](getting-started.md) · [Tuner](tuner.md)

The Forge handles material-based upgrades.

## Forge basics

The Forge uses a base item plus required materials. If the recipe matches, the station produces the configured upgraded result.

Common inputs include:

- a base item
- a forge core
- a crystal or other catalyst

Current recipes are available in the DawnMC Discord.
