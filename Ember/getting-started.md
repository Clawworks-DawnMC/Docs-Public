# Getting Started

[Player index](README.md) · [Tuner](tuner.md) · [Forge](forge.md)

DawnMC-Ember is a station-based upgrade system. You use special GUIs to convert or improve items instead of crafting everything in a normal table.

## Main systems

- **Tuner**: upgrades items through a timing-based resonance minigame
- **Forge**: combines a base item with required materials into a higher-tier result

## What players should expect

- recipe validity is server-configured
- some upgrades preserve item state from the source item
- success and failure outcomes depend on the station and recipe
- staff may limit access to specific commands or stations
