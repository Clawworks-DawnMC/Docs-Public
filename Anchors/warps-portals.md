# Warps & Portals

[Player index](../README.md) · [Anchors & Utility](anchors.md) · [Homes](homes.md) · [Random Teleport](rtp.md) · [TPA](tpa.md)

Warps are named server destinations. Portals are configured travel points that can be entered physically in the world or triggered from the command line.

## Warps

| Command | What it does |
|---|---|
| `/warp <name>` | Teleports you to a warp. |
| `/warp help` | Shows warp command help. |
| `/warps` | Lists all available warps. |
| `/warps list` | Lists all available warps. |
| `/warps help` | Shows warps command help. |

## Portals

| Command | What it does |
|---|---|
| `/portal <name>` | Uses a configured portal by name. |
| `/portal list` | Lists the portals you are allowed to use. |
| `/portal info <portal>` | Shows details for a portal you are allowed to access. |
| `/portal show <portal>` | Shows a temporary visual preview of a portal region. |
| `/portal help` | Shows portal command help. |

## Notes

- Some portals may require an additional server-defined permission before they can be used.
