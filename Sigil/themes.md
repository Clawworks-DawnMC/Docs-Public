# Themes

[Player index](README.md) · [Commands](commands.md) · [Profiles](profiles.md)

Sigil lets you switch your personal UI theme when you have permission to use that theme.

## Main commands

| Command | What it does |
|---|---|
| `/sigil theme` | Shows your current active theme. |
| `/sigil theme list [category]` | Lists themes you can use. |
| `/sigil set theme <theme>` | Sets your active theme. |
| `/sigil theme clear` | Clears your active theme and your overrides. |
