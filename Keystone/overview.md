# Overview

[Player index](README.md)

Keystone is intended to own non-travel utility systems that reside within a typical Essentials-style suite.

## Current status

There are no stable player-facing Keystone commands documented in this snapshot.

That means:

- no public command reference is exposed here yet
- any future public docs should be written from the implemented plugin, not only from the product spec
- players should use Anchors, Bloom, Ember, and Sigil docs for currently documented live systems
